# 🚀 UniExchange

A modern web-based project built using **Vite** for fast performance and smooth development experience.

---

## 📌 About the Project

**UniExchange** is a frontend-based web application designed to demonstrate a fast and efficient UI setup using Vite.
It focuses on simplicity, speed, and clean structure.

---

## 🛠️ Tech Stack

* ⚡ Vite
* 🟨 JavaScript
* 🌐 HTML5
* 🎨 CSS3

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/anmol0805/UniExchange.git
```

### 2. Navigate to the project folder

```bash
cd UniExchange
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```

---

## 🌐 Run Locally

After running the above command, open your browser and go to:

```
http://localhost:5173/
```
